# 🧠 AI Data Analyst Assistant

> **Transform raw data into actionable insights using the power of LLMs + Python + Interactive Visualization**

---

## 🚀 Overview

The **AI Data Analyst Assistant** is an end-to-end intelligent data analysis tool that combines **Large Language Models (LLMs)** with **Python-based analytics** to automate exploratory data analysis (EDA) and answer natural language queries on datasets.

Built with **Streamlit**, this application allows users to upload datasets and interact with them conversationally — just like talking to a real data analyst.

---

## ✨ Key Features

🔹 **📊 Automatic EDA**

* Data preview, missing values, and statistics
* Correlation heatmaps & distributions
* Clean and interactive visualizations using Plotly

🔹 **💬 Conversational Data Analysis**

* Ask questions in plain English
* AI generates Python code dynamically
* Executes analysis in real-time

🔹 **⚡ Smart Code Execution Engine**

* Safe execution sandbox
* Captures outputs, errors, and visualizations
* Auto-renders charts

🔹 **📈 Interactive Visualizations**

* Powered by Plotly (no static graphs)
* Dynamic and professional charts

🔹 **🧠 LLM Integration**

* Uses Anthropic Claude model for reasoning
* Generates optimized and clean analysis code

🔹 **📁 Multi-format Support**

* Upload CSV, XLSX, XLS files

🔹 **💾 Export Chat History**

* Download complete analysis conversation

## 🧩 How It Works

1. Upload your dataset 📂
2. App performs automatic EDA 📊
3. Ask questions like:

   * *"Show sales trend over time"*
   * *"Find correlation between features"*
   * *"Top 10 customers by revenue"*
4. AI:

   * Understands your query
   * Generates Python code
   * Executes it
   * Displays results instantly ⚡

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/ai-data-analyst.git

# Navigate to project folder
cd ai-data-analyst

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
